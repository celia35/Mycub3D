**Description**
Cub3D is a graphic design game, representing the inside of a maze from a first-person perspective. On of the goal is to understand the ray-casting. 

**Instruction**
The management of the windows has to remain smooth.
We have to display different wall textures that depending on wich side of the wall is facing(North, South, East, West).
The program must be able to set the floor abd ceiling colors to two different ones.
The program displays the image in a window:
	The left and right arrow keys of the keyboard allow the player to look left and right in the maze.
	The W, A, S, AND D allow the player to move the point of view through the maze.
	Pressing ESC must close the window.
	Clincking on the red cross on the window's frame must close the window and quit the program cleanly.

The program must take as a first argument a scene description file with the .cub extension.

Ecept for the map, each element must begin with its type identifier, followed by its specific in a strict order.
